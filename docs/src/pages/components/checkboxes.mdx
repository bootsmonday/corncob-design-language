# Checkboxes

Checkboxes are used to allow users to select one or more options from a set. They can be used in forms, settings, and anywhere else where multiple selections are needed.
Corncob checkboxes are standard HTML checkboxes styled to fit the Corncob design system.
They can be used in various states such as checked, unchecked, and disabled.
