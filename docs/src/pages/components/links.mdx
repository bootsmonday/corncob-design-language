## Links

Links are used to navigate between pages or sections of a website. They can be styled in various ways to indicate their purpose and importance.

### Basic Link

The most basic link is created using the `<a>` tag. It can be styled with CSS to change its appearance. Here’s an example of a basic link:

<div class="corn-panel">
  <a href="https://www.example.com">Visit Example</a>
</div>

<CodeSample id="basic-link-code">
  ```html
  <a href="https://www.example.com">Visit Example</a>
  ```
</CodeSample>

### Stand-alone Link

A stand-alone link is a link that is not part of a larger block of text. It can be styled to look like a button or other interactive element. Here’s an example of a stand-alone link:
Add the `corn-link` class to the `<a>` tag to apply the link styles.

<div class="corn-panel">
  <div>
    <a href="#example-link" class="corn-link">
      Forgot Username or Password?
    </a>
  </div>

  <div>
    <a href="#example-link" class="corn-link">
      Create a new account
    </a>
  </div>
</div>

<CodeSample id="stand-alone-link-code">
  ```html
  <div>
    <a href="#example-link" class="corn-link">
      Forgot Username or Password?
    </a>
  </div>

  <div>
    <a href="#example-link" class="corn-link">
      Create a new account
    </a>
  </div>
  ```
</CodeSample>

### Link List

A link list is a collection of links grouped together in a list structure, typically used for navigation or related content. The `corn-link-list` class is applied to the `<ul>` element, and each link within the list uses the `corn-link` class on the `<a>` tags inside the list items.
<ul class="corn-link-list">
  <li><a href="#" class="corn-link">Link list item one</a></li>
  <li><a href="#" class="corn-link">Link list item two</a></li>
  <li><a href="#" class="corn-link">Link list item three</a></li>
</ul>
<CodeSample id="link-list-code">
  ```html
  <ul class="corn-link-list">
    <li><a href="#" class="corn-link">Link list item one</a></li>
    <li><a href="#" class="corn-link">Link list item two</a></li>
    <li><a href="#" class="corn-link">Link list item three</a></li>
  </ul>
  ```
</CodeSample>
