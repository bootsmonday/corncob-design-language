## Headers

Headers are used to provide structure and hierarchy to your content. They help users understand the organization of information and navigate through your application.
Corncob provides a simple and elegant design for headers, ensuring a consistent user experience across your application.

### AI Implementation Contract

This component's documented example is canonical. Use this code before adding any customization.

AI should preserve these direct children of `corn-header` in this order:

1. `corn-company`
2. `corn-header--title`
3. `corn-header--nav`
4. `corn-header--actions`

Use the Corncob grid around the header for page layout when needed. Do not introduce extra wrappers inside `corn-header` unless the documentation explicitly shows them.
