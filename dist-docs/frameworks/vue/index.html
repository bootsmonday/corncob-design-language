Vue Version

Vue wrappers can follow the same contract as React wrappers: classes are generated from props while all design behavior remains in shared Corncob CSS.

Wrapper Pattern

<script setup>
import "@corncob/edl/style.css";
import { CcButton } from "@corncob/edl-vue";
</script>

<template>
  <CcButton size="sm">Save</CcButton>
  <CcButton variant="secondary" size="sm">Cancel</CcButton>
  <CcButton variant="danger" size="sm">Delete</CcButton>
</template>
