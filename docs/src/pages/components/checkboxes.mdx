## Checkboxes

Checkboxes are used to allow users to select one or more options from a set. They can be used in forms, settings, and anywhere else where multiple selections are needed.
Corncob checkboxes are standard HTML checkboxes styled to fit the Corncob design system.
They can be used in various states such as checked, unchecked, and disabled.

### Indeterminate Checkboxes

Indeterminate checkboxes are used to indicate a state that is neither checked nor unchecked.
This is commonly used when a checkbox represents a parent selection that has some, but not all, of its child checkboxes selected.
The indeterminate state is set via JavaScript on the checkbox input element using the `indeterminate` property.

<fieldset class="corn-form--item corn-checkbox-group">
  <div class="corn-checkbox">
    <input type="checkbox" id="example-indeterminate" name="example-indeterminate" />
    <label for="example-indeterminate">Indeterminate checkbox</label>
  </div>
