Corncob is a design language built with the goal of being framework-neutral, portable, and accessible. It provides a set of design tokens, CSS styles, and JavaScript behaviors that can be used to create consistent and user-friendly interfaces across different platforms and frameworks.

Corncob believes in utilizing as much built-in browser functionality as possible, with a focus on HTML and CSS first, and Web Components and JavaScript as needed. The design language includes a 10-step color palette to ease accessibility and provides design tokens for primitives, categorical elements, and components.

Corncob wants to be boring, but in a good way. The goal is to provide a solid foundation for building interfaces without adding unnecessary complexity or flair. By keeping the core package vanilla and portable, Corncob aims to be a reliable and flexible design language that can be easily integrated into any project.


### Goals

- Keep the core package vanilla and portable
- Utilize as much built-in browser functionality as possible.
- HTML and CSS first, Web Components and JavaScript as needed.
- 10 Steps Color Palette to ease accessibility
- Design Tokens from Primitive, Categorical, Components
