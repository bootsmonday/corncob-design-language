React Version

React wrappers should stay thin. Map React props to Corncob class names and keep style logic in the core CSS package.

Wrapper Pattern

import "@corncob/edl/style.css";
import { CcButton } from "@corncob/edl-react";

export function ToolbarActions() {
  return (
    <div>
      <CcButton size="sm">Save</CcButton>
      <CcButton variant="secondary" size="sm">Cancel</CcButton>
      <CcButton variant="danger" size="sm">Delete</CcButton>
    </div>
  );
}
