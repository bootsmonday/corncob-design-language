## About Corncob Design Language

Corncob Design Language is a design system created by BootsMonday to provide a consistent and cohesive user experience across all of our products. It is built on the principles of simplicity, accessibility, and scalability, and it includes a wide range of components, patterns, and guidelines to help designers and developers create beautiful and functional interfaces.

### Our Story

Corncob Design Language was born out of a need for a unified design system that could be used across all of our experiments. We wanted to create a system that would not only provide a consistent look and feel but also be flexible enough to accommodate the unique needs of each product. After months of research, design, and development, we launched Corncob Design Language in early 2026 to become the foundation for all of our future experiments.

### Our Tools
Corncob Design Language is built using a combination of modern web technologies, including HTML, CSS, and JavaScript. We use a variety of tools and frameworks to help us create and maintain our design system, including:

- **[Vite](https://vitejs.dev/)**: A fast and modern build tool that allows us to develop and build our design system efficiently.
- **[Astro](https://astro.build/)**: A modern static site generator that allows us to build fast and efficient documentation for our design system.
- **[Jest](https://jestjs.io/)**: A popular testing framework that we use for unit testing our components.
- **[BackstopJS](https://github.com/garris/BackstopJS)**: A visual regression testing tool that helps us ensure the visual consistency of our components across different browsers and devices.
- **[Axe Core](https://www.deque.com/axe/)**: An accessibility testing tool that we use to identify and address accessibility issues in our components.
- **[GitHub Actions](https://github.com/features/actions)**: A powerful automation tool that we use to automate our testing and deployment processes.
- **[Copilot](https://github.com/features/copilot)**: An AI-powered code assistant that helps us write and maintain our codebase more efficiently.
- **[Grok](https://grok.com/)**: An AI-powered code search and navigation tool that helps us quickly find and understand code across our codebase. Also acts as check against what Copilot suggests.
