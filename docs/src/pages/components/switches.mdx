## Switches

Switches allow users to select one option from a set of mutually exclusive options. They are typically used for settings or preferences where only one option can be active at a time. Corncob switches are styled checkboxes that visually resemble toggle switches. They are built using native `<input type="checkbox">` elements for accessibility and functionality, with custom CSS for the switch appearance.
